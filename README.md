#### Item Filter:
* Home -> Show / Hide Compass
* Num 0 -> Filter Throwables
* Num 1 -> Filter Attachments
* Num 2 -> Filter Scopes
* Num 3 -> Filter Ammo
* Num 4 -> Filter Weapons
* Num 5 -> Filter Level 2 Gear          
* Num 6 -> Filter Meds

#### Icon Toggles
* F5 -> Toggle Vehicle Names
* F6 -> Toggle Vehicles
* F11 -> Toggle View Line
* Insert -> Grid

#### Zooms:
* Num 7 -> Scouting
* Num 8 -> Scout/Loot
* Num 9 -> Looting
* F9 ->  Camera Zoom ++
* F10 -> Camera Zoom --
